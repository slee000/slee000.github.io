Robust Online Model Identification for Versatile Robot Control Based on Self-Attention Learning


Prepared to submit.

The robustness of a machine learning-based control algorithm was demonstrated by using the algorithm to control a legged robot. In more detail, I constructed a test environment with the commercial quadruped robot that has 12 degrees of freedom (DOFs). This robot was controlled by each motor position command in real-time. The command was generated by Python codes with the machine learning-based control algorithm and was transmitted to the robot through user datagram protocol (UDP) communication. This communication was implemented by libraries (1), (2). The current robot states were also transmitted to the computer through UDP communication by Unitree libraries.
